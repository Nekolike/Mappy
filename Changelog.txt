Update 18.01.2020 - Version 1.0

- You can now remove any category / keyword with a right-click on the category / keyword. There's also a new "-" button at the end of each category which removes the last keyword. There's also a new "-" button at the bottom of your categories which removes the last category (With all the keywords in them). 
- You can now save multiple overlays. Create a new overlay with the "New Overlay" option and hit "Save As New Overlay" at the end to create a new *.ini with your new overlay. 
- You can now load overlays. Click "Load Overlay" and select one of your *.ini files which you'd like to use.
- Informations about Mappy like it's position, the currently selected .ini and the hotkey to show/hide it can now be found in the "Settings.ini"

Update 13.01.2020 - Version 0.7.5

- You can now add categories & keywords on the fly. Check out the new "+"-buttons on your overlay. No need to set up your whole overlay again just because you forgot one keyword/category!
- Renamed "Create Keywords" to "New Overlay"

Update 13.01.2020 - Version 0.7.2

- You can add as many categories as you like with as many keywords as you like! But keep the visual cluster in mind.

- Saving your categories won't highlight the overlay-window anymore to prevent confusion. Same with adding keywords.
- Fixed a bug where not all keywords were displayed under certain circumstances. 

Update 12.01.2020 - Version 0.7.0 + 0.7.1

- Edit-fields which have no value will now be grayed out to reduce confusion
- Fixed a bug which would result in ERROR-buttons when canceling the create-process
